# Moltbot Lite Analytics

A lightweight, privacy-friendly analytics starter to replace GA4 for simple needs.

## Why

A recurring pain point: **GA4 fatigue**. Builders want something that’s:
- simple
- trustworthy
- privacy-friendly
- easy to self-host

## What’s in this repo

- Next.js (App Router) + TypeScript + Tailwind
- `/api/collect` ingestion endpoint
- Drizzle schema for an `events` table

## Local dev

```bash
pnpm install
pnpm dev
```

## Environment

You will need a Postgres `DATABASE_URL`.
